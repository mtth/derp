Derp
====

Minimal CLI chat server / client.


Quickstart
----------

To build:

.. code-block:: bash

  $ make

To start the server:

.. code-block:: bash

  $ ./bin/herp 4848

To start a client and connect it to the server:

.. code-block:: bash

  $ ./bin/derp 127.0.0.1 4848 alice
